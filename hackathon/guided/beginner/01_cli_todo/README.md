# Beginner 01 – CLI To‑Do List

Build a simple command‑line to‑do list manager.

## Requirements

* **add** a task
* **list** all tasks
* (Optional) **done** to mark complete
* Persist tasks to `tasks.json`

## Example

```bash
python todo.py add "Buy milk"
python todo.py list
```

## Copilot Tips

* Draft functions with docstrings; accept Copilot suggestions.
* Use built‑in `json` + `pathlib`; no external libs required.
